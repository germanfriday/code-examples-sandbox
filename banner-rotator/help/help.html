jQuery Banner Rotator

help guide

Installation

To install the banner rotator, first include the style sheet inside the <head> tag: 

<link rel="stylesheet" type="text/css" href="css/banner-rotator.css" />

Next, include the jQuery's library file, the easing plugin file, and the banner rotator's script file inside the <head> tag: 

<script type="text/javascript" src="js/jquery-1.10.2.min.js"></script>
<script type="text/javascript" src="js/jquery.easing.1.3.min.js"></script>
<script type="text/javascript" src="js/jquery.banner-rotator.js"></script>

This completes the installation process, next you'll have to specify the HTML markup.


HTML Markup

To insert a banner rotator for your page, specify the banner rotator's markup inside the <body> tag. The following is an example of a basic markup: 

<div id="rotator1" class="banner-rotator">
    <ul class="br-slides">
        <li>
            <img src="images/photo1.jpg"/>
        </li>                    
        <li> 
            <img src="images/photo2.jpg"/>
        </li>
        <li> 
            <img src="images/photo3.jpg"/>
        </li>
    </ul>
</div>

From the example, the <div> tag on line 1 represents the banner rotator container. The <div> tag should have the required class banner-rotator but can have any id value. For this example, we use an id of rotator1 which will be referenced in the activation function call.

Also of importance is the <ul> tag with class br-slides. This <ul> tag will hold all the <li> tags, with each <li> tag representing an individual slide.


Images

There are two ways to add an image to the banner rotator. One is using a regular <img> tag and the other is using an <a> tag with a link to the image. You can mix the two methods for each image in the banner rotator.

For the first method, just insert an <img> tag inside the <li> tag: 

<li>
    <img src="images/photo1.jpg" title="photo 1"/>
</li>

This is the simpler of the two methods but the image is downloaded immediately versus when needed. Also, the title value will be used for the thumbnail's text and the same image will be used for the thumbnail's image.

For more information on setting the thumbnail type, refer to the navigation control options.


For the second method, to add an image using a link, insert an <a> tag inside the <li> tag: 

<li>
    <a href="images/photo1.jpg">photo 1</a>
</li>

This method will reduce the initial download time since the image is downloaded as needed instead of all at once. Also, the text content inside the <a> tag is used as the thumbnail's text.

To have a thumbnail image instead of text, insert an <img> tag inside the <a> tag like the following: 

<li>
    <a href="images/photo1.jpg">
        <img src="images/thumb1.jpg" title="photo 1"/>
    </a>
</li>

Text Tooltip

By default, the tooltip will use the same text as its thumbnail's text. You can however enter a completely different text by embedding a tag with class tt-html with the tooltip content inside: 

<li>
    <a href="images/photo1.jpg">photo 1</a>
    <div class="tt-html">
    	lorem ipsum
    </div>
</li>

This is especially useful when you want html content for the tooltip instead of just plain text.

For more information on setting the tooltip type, refer to the navigation control options.


Image Tooltip

By default, the tooltip will use the thumbnail's image as its tooltip image. You can however embed a completely different tooltip image by inserting an <img> tag with class tt-img: 

<li>
    <a href="images/photo1.jpg">photo 1</a>
    <img src="images/tooltip1.jpg" class="tt-img"/>
</li>

For more information on setting the tooltip type, refer to the navigation control options.


Links

To add a link for the slide, use the data-link attribute for the <li> tag. You can also specify an optional link target by using the data-target attribute: 

<li data-link="http://www.google.com" data-target="_blank">
    <a href="images/photo1.jpg">photo 1</a>
</li>

Image Transition

You can configure the transition behavior of each slide. To customize these options, set the following data attributes for the <li> tag: 

<li data-effect="fade" data-duration="500" data-easing="linear" data-delay="5000">
    <a href="images/photo1.jpg">photo 1</a>
</li>

The available options are:

  • data-effect - the image's transition effect. (see image effects for available values)
  • data-duration - the transition's duration in milliseconds.
  • data-easing - the transition's animation easing.
  • data-delay - the delay in milliseconds before the image changes.

Alternatively, you can use the shorthand attribute data-transition instead to specify all four options at once: 

<li data-transition="fade 500 linear 5000">
    <a href="images/photo1.jpg">photo 1</a>
</li>

Caption Layers

You can add caption layers on top of the slide's image. These caption layers can hold most html content as well as plain text. To add a caption layer, insert any normal tag with its content inside: 

<li>
    <a href="images/photo1.jpg">photo 1</a>
    <h1>
        Lorem ipsum dolor sit amet, consectetur adipiscing elit. 
        Sed convallis justo arcu. Fusce adipiscing condimentum aliquam.
    </h2>
</li>

To add multiple caption layers for the slide, insert each layer tag one after another: 

<li>
    <a href="images/photo1.jpg">photo 1</a>
    <h1>
        Lorem ipsum dolor sit amet, consectetur adipiscing elit. 
        Sed convallis justo arcu. Fusce adipiscing condimentum aliquam.
    </h1>
    <h2>
        Quisque ultrices, dui a pellentesque blandit, nisl velit posuere enim.
        Nulla commodo urna non velit venenatis id aliquet lorem tincidunt.
    </h2>
    <div>
        Fusce fringilla cursus purus dignissim elementum.  
        Phasellus vehicula ligula eu justo tempus eget adipiscing justo porttitor.
    </div>
</li>

You can use virtually any tags to represent a caption layer. Some examples are <span>, <div>, <p>, and <h1> thru <h6>.


Layer Style

Normally, you would want to adjust each caption layer like its top, left, width, and height properties. In order to do so, use the style attribute and set any necessary properties: 

<li>
    <a href="images/photo1.jpg">photo 1</a>
    <h1 style="top:10px; left:25px; width:300px; height:auto; padding:5px; background-color:#00f; border:1px solid #000">
        Lorem ipsum dolor sit amet, consectetur adipiscing elit. 
        Sed convallis justo arcu. Fusce adipiscing condimentum aliquam.
    </h1>
</li>

You can set almost any properties as you would for any normal tag.
Some useful style properties are:

  • top - the top position of the layer.
  • left - the left position of the layer.
  • bottom - the bottom position of the layer. (If top is set, bottom will be ignored.)
  • right - the right position of the layer. (If left is set, right will be ignored.)
  • width - the width of the layer. (Use auto for auto width.)
  • height - the height of the layer. (Use auto for auto height.)
  • padding - the padding of the layer.
  • border - the border style of the layer.
  • color - the color of the text.
  • background-color - the background color.

Layer Transition

You can configure the transition behavior of each layer. To customize these options, set the following data-* attributes for the layer tag: 

<li>
    <a href="images/photo1.jpg">photo 1</a>
    <h1 data-effect="fade" data-duration="1000" data-easing="swing" data-delay="100">
        Lorem ipsum dolor sit amet, consectetur adipiscing elit. 
        Sed convallis justo arcu. Fusce adipiscing condimentum aliquam.
    </h1>
</li>

The customizable options are:

  • data-effect - the layer's transition effect. (see layer effects for available effects)
  • data-duration - the transition's duration in milliseconds.
  • data-easing - the transition's animation easing.
  • data-delay - the delay in milliseconds before the transition starts.

Alternatively, you can use the shorthand attribute data-transition instead to specify all four options at once: 

<li>
    <a href="images/photo1.jpg">photo 1</a>
    <h1 data-transition="fade 1000 swing 100">
        Lorem ipsum dolor sit amet, consectetur adipiscing elit. 
        Sed convallis justo arcu. Fusce adipiscing condimentum aliquam.
    </h1>
</li>

Layer Transition Out

You can configure the transition out behavior of each layer. To customize these options, set the following data-* attributes for the layer tag: 

<li>
    <a href="images/photo1.jpg">photo 1</a>
    <h1 data-effect-out="moveDown" data-duration-out="800" data-easing-out="linear" data-delay-out="1000">
        Lorem ipsum dolor sit amet, consectetur adipiscing elit. 
        Sed convallis justo arcu. Fusce adipiscing condimentum aliquam.
    </h1>
</li>

The customizable options are:

  • data-effect-out - the layer's transition out effect. (see layer effects for available effects)
  • data-duration-out - the transition out duration in milliseconds.
  • data-easing-out - the transition out animation easing.
  • data-delay-out - the delay in milliseconds before the transition out starts.

Alternatively, you can use the shorthand attribute data-transition-out instead to specify all four options at once: 

<li>
    <a href="images/photo1.jpg">photo 1</a>
    <h1 data-transition-out="moveDown 800 linear 1000">
        Lorem ipsum dolor sit amet, consectetur adipiscing elit. 
        Sed convallis justo arcu. Fusce adipiscing condimentum aliquam.
    </h1>
</li>

Activation & Configuration

Once the HTML markup is specified, the final step is to activate it to get it running. To activate the banner rotator, simply use the following script in your page: 

<script type="text/javascript">
$(window).load(
   function() {
      $("#rotator1").bannerRotator();
   }
);
</script>

The code snippet activates the banner rotator by using the id of the banner rotator's <div> container tag and applying the activation function on it. Also, make sure the activation function call is within the $(window).load() function.


The banner rotator has many configurable options to suit virtually any needs. In order to configure the banner rotator, pass in a list of the options to the activation function call. The following example activates the banner rotator, but this time with the option width and height specified. 

<script type="text/javascript">
$(window).load(
    function() {
        $("#rotator1").bannerRotator({
            width:1200,
            height:400
        });
    }
);
</script>

Please note any options not specified for the function call will take on its default value. For more information on all available options, please refer to the Options tables.

After activation and configuration, your banner rotator is ready to run.
