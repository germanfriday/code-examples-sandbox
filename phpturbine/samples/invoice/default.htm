Dynamic Invoice Sample
(PDF)
 

This sample shows how Turbine can be used to automate the process of creating PDF invoices. The invoice is dynamically generated, based on a template in which dynamic text is filled by using Turbine variables. Dynamic document generation into PDF format can easily be implemented by using Turbine 7.

On this sample the invoice is generated from the following fields:

Invoice Nr:
Bill To:
Ship To:
Product Description: Qty: Price: Amount:
Total: 
   


The Flash template used for this example is invoice.swf.

   
Note: if the generated invoice fields appear empty, it could be caused by known issues between Internet Explorer, Apache 2.0.43 and the PHP engine version 4.3.1, if this is the case, upgrading Apache and/or the PHP engine to more recent builds will fix the problem.
