A) HTML Structure - top

The Index.html tag has an id of #home-page. every other page has an id of #sub-page.
There are 4 main divs: #header, #content, #footer-wrapper. A Div with a class of "container" is used throghout the site to center the content. Div with an id of "header" contains such areas as logo and top menu. The top menu is automatically positioned to the right of the container, so you can easily add, edit, delete its items.

B) CSS Files and Structure - top

There are three css files necessary for customization.

  1. "reset.css" is a generic reset file.
  2. "ie7.css" contains all the fixed styles for ie7.
  3. "style.css" (default color style) is the main css file including the elements that were overriden.

The style.css file is separated into sections:

/** Typography **/

/** Header **/

/** Content **/

NOTE: If you would like to edit a specific section of the site, simply find the appropriate label in the CSS file, and then scroll down until you find the appropriate style that needs to be edited. There are still some css files not noted above. I did not include them because you don't have to modify them. The 3 files above are enough to make the changes you need.

C) JavaScript - top

This theme imports 14 Javascript files. All located in the "js" folder.

  1. jquery-1.3.2.min.js - jQuery Library.
  2. jquery.prettyPhoto.js - Script for lightbox plugin.
  3. jquery.cycle.all.js - Cycle plugin.
  4. jqFancyTransitions.js - Fancy-Transitions plugin.
  5. jquery.form.js - Jquery form plugin.
  6. init_form.js - The script the makes the form in the contact page work.
  7. superfish.js - The superfish dropdown menu script.
  8. cufon-yui.js - Cufon script.
  9. Delicious.font.js - Font for cufon script.
  10. jquery.easing.js - The easing plugin.
  11. coin-slider.js - The Coin slider plugin.
  12. ui.core.min.js - The Jquery UI core.
  13. ui.tabs.min.js - Jquery tabs.
  14. bluz.js - Cutom script.

D) PSD Files - top

The PSD folder contains sliced psd files for both skins. In total there are 5 psd files:

  1. robust-home.psd - The Home-Page layout - Day skin.
  2. robust-subpage.psd - The Home-Page layout - Night skin.
  3. nav-right.psd - The right part of the top navigation.
  4. nav-left.psd - The left part of the top navigation.
  5. logo.psd - logo for easy edit.

If you'd like to change the logo, open "logo.psd", make the necessary adjustments, and then save the file as "logo.png".
